# Auxhack18

#### Modes

##### Idle:
Get: http://localhost:8079/avatar/idle/  
Avatar im Idle Mode (schaut nur geradeaus)

##### Follow:
Get: http://localhost:8079/avatar/follow/  
Avatar im Follow Mode (folgt dem erkannten Gesicht auf der Kamera)  
(Erfordert ein das zusätzliche Projekt "facedetect")

##### Speak:
Get: http://localhost:8079/avatar/speak/  
Avatar im Speak Mode (Sprechblase wird angezeigt)

##### Multiply:
Get: http://localhost:8079/avatar/multiply/  
Avatar wird mehrfach angezeigt

##### Chicken:
Get: http://localhost:8079/avatar/chicken/  
Als Avatar wird ein Hühnchen angezeigt
